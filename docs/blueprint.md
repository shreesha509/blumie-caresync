# **App Name**: MoodLight

## Core Features:

- Mood Input: Mood Input: A single screen with a text input field for describing mood, and a color picker to choose a mood color. Pressing a button submits the chosen mood and color to the user interface.
- Mood Display: Display mood as combination of a descriptive label from text and colored background corresponding to color picker
- Psych Game Placeholder: Placeholder screen for a psychological game include some basics games (Q&A include MCQ).
- Data View Placeholder: Placeholder screen for synced biometric data display. Links out to google fit.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6), inspired by feelings of serenity and peace. A calming hue appropriate for an app focused on mood.
- Background color: Very light blue (#E3F2FD), visibly related to primary blue but significantly desaturated, so the blue is a calming background.
- Accent color: Soft green (#81C784), an analogous color that suggests balance and well-being, clearly distinct in saturation and brightness to provide good contrast for key UI elements.
- Font: 'PT Sans', a humanist sans-serif, suitable for both headlines and body text.
- Use simple, clean icons to represent different moods or data types.
- Employ a clean, single-column layout for ease of use and clarity.
- Use subtle animations when transitioning between screens.
